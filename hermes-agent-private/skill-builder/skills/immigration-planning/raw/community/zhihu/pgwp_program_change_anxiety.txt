  问题：在加拿大读书换了专业，PGWP还能申请吗？我现在非常焦虑

COMMENT [rank=1, length=31]:
提问者：匿名用户 | 关注：342人 | 2024-12-15

COMMENT [rank=2, length=161]:
我是2023年9月入学，原来读的是计算机科学（CS），2024年3月换成了商科（Business Administration）。现在2024年11月IRCC宣布PGWP需要专业在eligible list上，Business Administration很多学校是不在列表里的。我是不是彻底没有PGWP资格了？有没有救？

COMMENT [rank=3, length=43]:
先说结论：你的情况很可能仍然有资格申请PGWP，因为grandfathering条款。

COMMENT [rank=4, length=113]:
IRCC 2024年11月的新政策原文是：2024年11月1日之前已经入学的学生，按照旧政策执行，不受新的专业要求限制。关键日期是"入学日期"，不是"换专业日期"。你是2023年9月入学，符合grandfathering条件。

COMMENT [rank=5, length=113]:
但这里有一个重要的细节：你换专业的时间（2024年3月）在2024年11月之前。IRCC目前的解释是：如果你在新政策生效（2024年11月1日）之前已经在读，且一直保持full-time学籍，则不受新政策影响，即使你换了专业。

COMMENT [rank=6, length=137]:
实际操作建议：
1. 保留所有入学证明（2023年9月的enrollment letter）
2. 保留换专业批准文件（showing it was before Nov 1, 2024）
3. 毕业前向DSO确认PGWP申请资格
4. 如有疑虑，在毕业前3个月咨询移民律师

COMMENT [rank=7, length=77]:
这个问题触及了2024年PGWP改革里最混乱的部分：有很多学生在改革公告之后（2024年11月后）才知道这个政策，然后回头看自己的情况时产生了很大的焦虑。

COMMENT [rank=8, length=82]:
我想说一个更重要的观察：PGWP政策本身的稳定性在下降。2024年11月调整，2025年6月又调整（这次移除了约178个专业、新增了119个）。未来的调整不可预测。

COMMENT [rank=9, length=128]:
这意味着：依赖PGWP作为唯一移民工具的风险在上升。更稳健的策略是：入学时就选择在IRCC list上的专业，同时在学期间就开始积累加拿大工作经验（通过合法的on-campus工作或co-op），这样即使PGWP政策再变，你在加拿大的工作经验已经在积累了。

COMMENT [rank=10, length=148]:
我想分享一个真实案例。我的同学，2022年入学读hospitality management，一个在2025年6月被移除列表的专业。她不在grandfathering保护范围内（2022年入学，新政策要求是2024年11月前入学），但她的PGWP申请是在新政策公布之前提交的，IRCC已经批准了。

COMMENT [rank=11, length=50]:
教训：如果你有资格申请PGWP，早申请比晚申请风险更低，因为规则可能还会变。不要等毕业后很久才申请。

COMMENT [rank=12, length=188]:
另外关于一个常见误区：很多学生以为PGWP有效期等于学习时间，比如读了2年就给2年PGWP。实际上，PGWP有效期取决于学习时间 + 专业类别：学习8个月以下给最多1年；8个月到2年给等同学习时间；2年以上给最长3年。但2024年新政策之后，不在eligible list的专业，有效期上限变为2年，无论你读了多久。这个变化影响了很多读了3年专业的学生原本预期的3年PGWP。
