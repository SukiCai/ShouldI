  标题：H1B抽签第三次没中，有没有过来人说说怎么办？

COMMENT [rank=1, length=41]:
版块：美国工作签证 | 时间：2025-03-22 | 回复：89 | 赞：156

COMMENT [rank=2, length=97]:
楼主是SDE在硅谷某大厂，工资L4 base 17万，已经抽了三年H1B都没中，公司说可以帮转L1但需要先在加拿大office工作一年。现在非常迷茫，公司说的L1路径是不是坑？还有没有其他出路？

COMMENT [rank=3, length=142]:
L1-B（专业知识）转美国非常常见，不是坑。但需要注意：L1-B续签有5年上限，5年内必须转H1B或EB绿卡，否则只能再回海外一年才能申请新的L1。很多大厂用这个路径绕过H1B抽签，本质上是：加拿大工作1年 → 申请L1-B → 回美国继续工作 → 期间申L1-B转EB1B或EB2。

COMMENT [rank=4, length=116]:
我走了完全不同的路：转去了cap-exempt的研究型大学做researcher，H1B不用抽签，工资比大厂低30%，但是三年就拿到了绿卡，现在拿着绿卡又回大厂了，总体时间差不多，压力小很多。很多人不知道cap-exempt这条路。

COMMENT [rank=5, length=137]:
关于EB1A自我申请（杰出人才）：很多人以为这只适合诺贝尔奖得主，其实不是。评判标准有10项，满足3项即可。对于有一定发表、开源贡献、专业影响力的SDE来说，这条路比很多人想象的可行。主要优势：不需要PERM劳工证（省2-4年），可以自我申请，排期对印度以外的国籍几乎不排。

COMMENT [rank=6, length=162]:
关于移民中介的一个普遍误区：很多人认为律师帮你申请了就不会有问题。实际上律师负责的是文件合规，不负责策略优化。你的最优路径（L1 vs cap-exempt vs EB1A自我申请）需要你自己搞清楚，然后告诉律师执行哪一条。两个人去同一个律师，一个人问"帮我办L1"，另一个问"我的情况下最优路径是什么"，结果可能完全不同。

COMMENT [rank=7, length=130]:
我三年前跟楼主一样的情况，现在在加拿大，真的不后悔转换了赛道。加拿大Express Entry走CEC类别，拿到PR之后反而可以从容谈offer，薪资谈判筹码反而更大了，因为你不再是"必须依附这家公司的签证持有人"。身份焦虑消失之后，职业发展的清晰度完全不同。

COMMENT [rank=8, length=166]:
很多中国学生有一个心理障碍：觉得不拿美国绿卡就是"失败"，回加拿大或者回国是放弃。这个framing是错的。真正的问题是：10年后你在哪里的综合发展更好？美国绿卡只是众多路径之一，不是终点。很多同学在这个压力下做出了非最优决策（比如接受不好的公司因为它给H1B，或者一直等H1B而错过了加拿大EXPRESS ENTRY的最优窗口）。
