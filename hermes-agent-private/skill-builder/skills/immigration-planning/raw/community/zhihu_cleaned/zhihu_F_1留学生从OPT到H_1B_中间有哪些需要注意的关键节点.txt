作为过来人，我来梳理一下关键时间节点和常见误区。

COMMENT [rank=2, length=76]:
OPT的工作授权是EAD（Employment Authorization Document），需要提前60-90天申请，USCIS处理时间3-5个月。

COMMENT [rank=3, length=73]:
关键点：
- 申请OPT后有90天失业上限（整个OPT期间累计，不是单次）
- 换工作要在10天内通知DSO更新SEVIS
- 工作必须与专业相关

COMMENT [rank=4, length=33]:
条件：STEM学位（需在官方列表上）+ 雇主在E-Verify注册

COMMENT [rank=5, length=42]:
需要提交I-983 Training Plan，描述工作如何与你的STEM专业相关。

COMMENT [rank=6, length=57]:
关键误区：很多人以为换工作就需要重新申请，实际上STEM OPT是绑定到你身上的，但新雇主需要重新提交I-983。

COMMENT [rank=7, length=55]:
- 普通名额：65,000个
- 硕士以上（美国学位）：额外20,000个名额，先从这个池子抽，没中再进普通池

COMMENT [rank=8, length=59]:
重要：H-1B注册和STEM OPT不冲突。你可以在STEM OPT期间参加H-1B抽签，中了之后10月1日切换身份。

COMMENT [rank=9, length=107]:
如果3年内没中H-1B（概率约50-60%，三次尝试）：
1. 继续读书（F-1重置，新的OPT机会）
2. Cap-exempt雇主（大学、非营利研究机构）
3. 其他签证类别（O-1、TN等）
4. 转赴加拿大

COMMENT [rank=10, length=55]:
从OPT第一年就要和雇主谈H-1B意愿，不要等到STEM OPT最后一年才开始。大多数雇主需要时间走内部审批。

COMMENT [rank=11, length=93]:

补充一点：申请H-1B前确认雇主会支付premium processing（$2805），这样能在15个工作日内拿到结果，避免等待期间身份悬空。

COMMENT [rank=12, length=100]:
[2] (赞同数: 445)
关于90天失业上限，需要强调：这是整个OPT期间的累计总数，不是单次。所以如果第一份工作结束到第二份工作开始中间空了60天，你只剩30天buffer了。很多人不知道这个。

COMMENT [rank=13, length=84]:
[3] (赞同数: 312)
STEM OPT的雇主培训计划（I-983）要认真写，不要copy-paste通用模板。有人就是因为I-983写得太模板化收到了NOID。
