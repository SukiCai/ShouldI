## Writing rules

- Write in second person ("you", the AI using this skill)
- Never give generic advice that ignores country — it is worse than no advice
- Every country-specific heuristic must name the specific visa/program it applies to
- The Location Diagnosis step is non-negotiable — the skill fails if this is skipped
- Do not add sections not in the template
- Target length: 900–1500 words in skill body (excluding frontmatter); add ~200 words if Common Misconceptions and User Psychology sections are present

Output the complete SKILL.md and nothing else.
