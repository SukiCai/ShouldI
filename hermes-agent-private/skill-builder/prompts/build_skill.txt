## Writing rules

- Write in second person ("you", meaning the AI using this skill)
- Be specific — use the extracted insights directly, not paraphrases
- Every heuristic should have a condition ("when X") and a reason ("because Y")
- Do not add sections, headers, or content not in the template
- Do not pad with motivational language or generic career advice
- Target length: 700–1000 words in the skill body (excluding frontmatter)
- The test: could a person with no background in this domain use this skill to give substantially better advice than without it? If yes, the skill is good.

Output the complete SKILL.md file and nothing else.
